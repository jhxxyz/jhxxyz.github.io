TCP通信——C/S

客户端

  2. 连接服务器
  3. 发送/接收数据
  4. 关闭socket

1、建立套接字socket

定义：
建立套接字，返回套接字文件描述符

函数原型：

参数：

  • domain：协议族

    PF_INET / AF_INET -> Ipv4 网络协议
    PF_INET6 / AF_INET6 -> Ipv6 网络协议

  • type：套接字类型

    SOCK_STREAM -> 字节流套接字TCP
    SOCK_DGRAM -> 数据报套接字UDP

  • protocol：协议，0为默认协议

返回值：
成功：套接字文件描述符
失败：-1

连接服务器

函数原型：

参数：

  • sockfd：套接字文件描述符
  • addr：服务器地址和端口号
  • addrlen：地址长度

返回值：
成功：0
失败：-1

示例代码：初始化IP地址和端口号 –IPV4

发送/接收数据

函数原型：

参数：

  • sockfd：套接字文件描述符
  • buf：发送缓冲区
  • len：发送数据长度
  • flags：标志位，默认为0

返回值：

  • 成功：发送的字节
  • 失败：-1

服务器

1、绑定bind
2、监听listen
3、等待连接accept
4、接收数据recv

绑定bind

定义：
绑定自己的IP地址和端口号

函数原型：

参数：

  • sockfd：套接字文件描述符
  • addr：自己的IP地址和端口号
  • addrlen：地址长度

返回值：

  • 成功：0
  • 失败：-1

绑定IP与端口号的结构体（16个字节）： 使用 man 7 ip 可以查看以下结构体

监听listen

定义： 创建一个未连接队列，同时最多连接的客户端总数 backlog。

函数原型：

参数：

  • sockfd：套接字文件描述符
  • backlog：同时最多支持连接上来的客户端总数

返回值：

  • 成功：0
  • 失败：-1

等待连接accept

定义： 等待客户端连接

函数原型：

参数：

  • sockfd：套接字文件描述符
  • addr：客户端的IP地址和端口号
  • addrlen：地址长度

返回值：

  • 成功：已连接的客户端套接字文件描述符
  • 失败：-1

说明：
listen()与accept()是成对出现的，listen()用来创建一个未连接队列，accept()用来接收客户端连接。

接收数据recv

函数原型：

参数：

  • sockfd：套接字文件描述符
  • buf：接收数据存储的缓冲区
  • len：接收数据的大小
  • flags：标志位，一般为0

返回值：

  • 成功：接收到的字节数
  • 失败：-1
  • 0：客户端断开连接
